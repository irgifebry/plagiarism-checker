Plagiarism Checker & AI Detector

Open-source tool untuk deteksi plagiarisme dan AI-generated text.

Features

  • Plagiarism Detection — Real-time web scraping via Google & ddgr.
  • AI Detection — Statistik advanced (TTR, Burstiness, N-gram Diversity, Hapax Ratio).
  • Multi-Format — DOCX, PDF, TXT.
  • Agent-Ready — Autonomously runnable oleh AI Agent.
  • Zero API Cost — 100% offline, local database matching.

Quick Installation

# Setup Venv
python3 -m venv ~/venv/plagiarism-checker
source ~/venv/plagiarism-checker/bin/activate

# Install Dependencies
pip install pymupdf python-docx requests beautifulsoup4
sudo apt install ddgr -y

How It Works

  1. Extract Text — Convert DOCX/PDF/TXT ke clean text.
  2. AI Detection — Analisis statistik (TTR, N-gram, Variance, Hapax).
  3. Plagiarism Check — Chunk text, search via ddrr & Google, calculate similarity.
  4. Report — Generate Markdown report dengan hex highlights.

Usage

# Extract text
./scripts/extract_text.py document.docx > /tmp/text.txt

# Analyze
./scripts/analyze.py < /tmp/text.txt

# Output: Markdown report dengan plagiarism & AI scores

Full usage: GitHub README

Limits & Caveats

  • Web Access Only — Public data (Google/ddgr). No Turnitin, iThenticate access.
  • Statistical AI Detection — 85-95% accuracy (bukan 100%).
  • Rate Limited — Google scraping delay 3-5s per chunk.
  • Python 3.10+ required.
